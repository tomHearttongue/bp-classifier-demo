Privacy Policy

Last updated: July 15, 2026

This is a portfolio demonstration. Do not enter confidential, personal, proprietary, regulated, or production company data.

What This Demo Is

AI-Governed Internal Support Orchestrator is a public proof-of-concept demo for source-aware internal support workflows. It is not a production support system, ticketing platform, HR system, finance system, legal intake system, or access-management system.

Information You May Provide

If you use the optional classifier flow, the demo may send the support-request text you enter and service catalog metadata to a classifier endpoint for intent classification.

Information You Should Not Provide

Do not enter sensitive or confidential information, including personal data, employee identifiers, private company policies, security information, financial details, legal matters, health information, credentials, secrets, or production support requests.

How Information Is Used

Submitted text is used only to demonstrate intent classification and workflow-routing concepts. The classifier is intended to identify a likely support service and return a structured rationale. It should not make approval, eligibility, access, legal, financial, or fulfillment decisions.

Third-Party Processing

If an OpenAI-powered classifier endpoint is enabled, request text and service catalog metadata may be processed by OpenAI through that endpoint. The static GitHub Pages site does not contain an OpenAI API key.

Data Retention

This static site does not intentionally store submitted request text. A deployed classifier endpoint or hosting provider may generate operational logs depending on its configuration. Do not use the demo with information that should not appear in logs.

Cookies And Tracking

This demo does not intentionally use analytics cookies. The browser may store the optional classifier endpoint URL and mode locally so the demo remembers your settings.

Contact
